These artifacts are copied verbatim from [Elizabeth Fawcett's lexile-determination-v2 project](https://github.com/eannefawcett/lexile-determination-v2):

- `model.h5`: trained Keras classifier checkpoint.
- `tokenizer.pickle`: serialized Keras `Tokenizer` used to build TF-IDF vectors.
- `labels.pickle`: label encoder mapping class indices to Lexile bands.
- `stopwords.pickle`: stopword list consumed by the original preprocessing pipeline.
